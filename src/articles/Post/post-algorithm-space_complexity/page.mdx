### 空间复杂度的定义

空间复杂度指的是：一个算法在运行过程中占用空间大小的量度，记作 $S(n) = O(f(n))$

空间复杂度虽然记作 $S(n)$, 我们平时仍然使用 上界 O 来表示

### 空间复杂度的使用

空间复杂度用于预估程序内存的使用大小

<Aside title="程序内存的影响因素">
  编译器的内存对齐，编程语言容器的底层实现等等这些都会影响到程序内存的开销。
</Aside>

在线上答题 (Online Judge) 的时候，经常能够看到 超出内存限制 这种错误，因为计算机的内存空间也不是无限的，需要工程师对软件运行时所使用的内存有一个**大体评估**，这都需要用到算法空间复杂度的分析，因此线上答题平台会设置一个内存损耗上限。

### 空间复杂度的计算

当随着 n 的变化，所需开辟的内存空间并不会随着 n 的变化而变化时,该算法的空间复杂度为 $O(1)$ ：

```ts
function addToN(n: number) {
  let int = 0;
  for (let i = 0; i < n; i++) {
    int++;
  }
}
```

当消耗空间和输入参数 n 保持线性增长，这样的空间复杂度为 $O(n)$ :

```ts
function addElementToArray(n: number) {
  let arr = [];
  for (let i = 0; i < n; i++) {
    arr[i] = i;
  }
}
```

以此类推。
