## 居中问题

本文主要讨论一个 CSS 经久不衰的问题: :confused:如何实现一个子级元素在父级元素同时满足水平居中与垂直居中呢？

这个问题有多个拓展的空间，这里我将分情况进行解答。

## flex 控制主轴与副轴方向元素

主轴居中： `justify-content: center`

副轴居中： `align-items: center`

```html
<style>
  .box {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 10px;
    height: 300px;
    background-color: rgb(224, 206, 247);
    border: 5px solid rebeccapurple;
  }
  .child {
    width: 100px;
    padding: 10px;
    background: rgba(255, 255, 255, 0.5);
    border: 1px solid black;
  }
</style>

<div class="box">
  <div class="child">子级元素1</div>
  <div class="child">子级元素2</div>
</div>
```

## margin 控制水平居中

使用 margin 控制水平居中有两个条件：

1. 父元素不能自适应宽度。
2. 子元素拥有明确的宽度 (具体的宽度值或者 fit-content)

然后只要给子元素添加 `margin: 0 auto` 即可， 在确认`writing-mode`为默认值时，使用 `margin-inline: auto` 也可。

```html
<style>
  .box {
    width: 100%;
    height: 300px;
    background-color: rgb(224, 206, 247);
    border: 5px solid rebeccapurple;
  }
  .child {
    width: fit-content;
    padding: 10px;
    margin-inline: auto;
    background: rgba(255, 255, 255, 0.5);
    border: 1px solid black;
  }
</style>

<div class="box">
  <div class="child">子级元素1</div>
  <div class="child">子级元素2</div>
</div>
```
